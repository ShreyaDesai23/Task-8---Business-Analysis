# Customer Churn Business Analysis

This project analyzes telecom customer churn using data analytics and machine learning.

## Objective
Identify factors causing customer churn and provide business recommendations.

## Techniques Used
- Data Cleaning
- Exploratory Data Analysis
- Correlation Analysis
- Logistic Regression

## Tools
Python, Pandas, Seaborn, Scikit-Learn, Google Colab

## Business Insights
- Customers with high monthly charges churn more
- New customers churn more frequently
- Senior citizens show higher churn rates

## Recommendations
- Offer loyalty discounts
- Improve onboarding for new customers
- Introduce flexible pricing plans
